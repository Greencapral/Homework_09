# Домашнее задание

**Цель:**
Закрепить навыки работы с CBV (Class-Based Views) и написания автотестов с использованием Pytest.

**Результат:**
Приложение интернет-магазина с функциональностью CRUD через CBV и тестами для ключевых функций.


## что реализовано
- Подключил кастомное приложение для пользователей
- Реализованы CBV:
    - ListView для отображения списка товаров.
    - DetailView для отображения деталей товара.
    - CreateView и UpdateView для добавления и редактирования товаров. 
    - DeleteView для удаления товара.
- Написаны тесты для моделей: 
    - операции создания
    - операции чтения
    - операции обновления и удаления записей.

## как все это запустить
- синхронизировать среду командой: **uv sync**
- запустить сервер командой: **python manage.py runserver**
- по адресу http://127.0.0.1:8000/ можно "подергать ручки"
- по адресу http://127.0.0.1:8000/admin/ (логин: admin пароль: 1) можно поковырять админку
- командой: **python manage.py gen_data** можно дополнить модели случайными данными
- командой: **python manage.py loaddata example_datas.json** можно загрузить в модели эталонные данные
- как надоест выйти остановить сервер комбинацией **CTRL-BREAK**
- запустить все тесты командой **pytest -vv**
